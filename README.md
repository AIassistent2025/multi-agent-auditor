# Multi-Agent Financial Auditor System

An experimental framework that preceded the production "Pulse-Audit" project. This repository contains the foundational logic for deploying cooperative AI agents designed to evaluate financial health.

## Overview
- **Agent Roles**: Financial Analyst, Research Specialist, Audit Reviewer.
- **Goal**: Coordinated aggregation of raw financial data to produce comprehensive risk assessments.
- **Tech Stack**: Python, CrewAI, LangChain, OpenAI.
